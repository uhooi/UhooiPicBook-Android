## Development

You can develop this project.

### Environment

- Android Studio: 4.1.3

### Configuration

- Architecture: MVVM
- Branching model: Git-flow

### Setup

1. Clone the project.

```shell
$ git clone https://github.com/uhooi/UhooiPicBook-Android.git
$ cd UhooiPicBook-Android
```

2. Open the project in Android Studio.
